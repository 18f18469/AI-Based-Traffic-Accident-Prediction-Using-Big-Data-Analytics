Project Overview
This project presents an AI-Based Traffic Accident Prediction System developed using Big Data Analytics and Machine Learning techniques. The system aims to predict traffic accident risk levels based on historical traffic data to support decision-makers in improving road safety
Objectives
Predict traffic accident
Technologies tools used 
Python
Flask Framework
Machine Learning (Scikit-learn)
MySQL Database
HTML, CSS, Bootstrap
